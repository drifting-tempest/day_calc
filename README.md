# Day Calculator

This  website allows you to calculate the number of days between 2 dates given in dd/mm/yyyy format!

## Features

1. Accommodation for leap year logic
2. Accommodation for different months having different days
3. Does NOT make use of the datetime module
4. Confetti appears when the number of days is calculated! (YAY)

## Learning Experience With:

1. Connecting JavaScript, Python, HTML, and CSS (took a LOT of debugging and fixing code)
2. Working with form responses
3. Writing the algorithm while accommodating for leap years, different numbers of days in different months
